Comando WC

Muestra el número total de líneas , palabras y caracteres del archivo texto.txt.



-c: Muestra el número de bytes del archivo



El comando wc en la terminal de Linux se utiliza para contar palabras, líneas y bytes en archivos de texto. Un ejemplo simple:
wc texto.txt / Esto te dará una salida como esta: 3 14 86 texto.txt / Así que en este ejemplo, el archivo "texto.txt" tiene 3 líneas, 14 palabras y ocupa 86 bytes en total.

-m: Muestra el numero de caracteres


-l: Muestra el número de lineas


-L: muestra el tamaño de la linea mas larga


-w: Muestra el recuento de palabras


Ejemplo

wc -c fichero.txt
